# TypeScript Pro Command

Execute TypeScript development task using the typescript-pro subagent.

## TypeScript Development Task

$ARGUMENTS

## Instructions

1. Analyze TypeScript development requirements:
   - Project setup and tsconfig
   - Type coverage goals
   - Build tool configuration
   - Framework specifications
   - Performance targets
   - Team expertise level

2. Delegate to the `typescript-pro` subagent to:
   - Apply advanced type patterns
   - Ensure 100% type coverage for public APIs
   - Optimize builds with project references
   - Implement full-stack type safety
   - Setup strict mode enforcement
   - Create type-safe utilities
   - Achieve >90% test coverage
   - Optimize bundle size

3. The typescript-pro will implement:
   - Advanced type patterns
   - Type system mastery
   - Full-stack type safety
   - Build optimization
   - Type-safe testing
   - Framework patterns
   - Performance tuning
   - Error handling types

4. Provide TypeScript implementation including:
   - Type-safe code
   - Advanced patterns usage
   - Type definitions
   - Test suite
   - Build optimization results
   - Performance metrics
   - Type coverage report
   - Documentation

## Output Format

Output TypeScript development work in Japanese, following the typescript-pro subagent's response format.

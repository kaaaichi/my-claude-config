# Technical Writer Command

Create technical documentation using the technical-writer subagent.

## Documentation Task

$ARGUMENTS

## Instructions

1. Analyze documentation requirements:
   - Document type and purpose
   - Target audience
   - Existing documentation
   - Key topics to cover
   - Preferred format and tools
   - Publication timeline

2. Delegate to the `technical-writer` subagent to:
   - Plan information architecture
   - Create comprehensive documentation
   - Ensure technical accuracy
   - Provide practical examples
   - Include visuals and diagrams
   - Optimize for searchability
   - Write in clear, concise language
   - Follow style guidelines

3. The technical-writer will create:
   - Well-structured documentation
   - Clear explanations with examples
   - Consistent terminology
   - Visual aids and diagrams
   - Step-by-step instructions
   - Troubleshooting guides
   - Best practices
   - SEO-optimized content

4. Provide documentation including:
   - Complete documentation pages
   - Code examples
   - Screenshots/diagrams
   - Quick reference guides
   - User guides
   - API documentation
   - Troubleshooting section
   - Accessibility review

## Output Format

Output technical documentation in Japanese, following the technical-writer subagent's response format.

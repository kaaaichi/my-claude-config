# Frontend Developer Command

Execute frontend development task using the frontend-developer subagent.

## Frontend Development Task

$ARGUMENTS

## Instructions

1. Analyze frontend development requirements:
   - UI/UX specifications
   - Component requirements
   - Design system specifications
   - Accessibility requirements
   - Performance targets
   - Browser support needs

2. Delegate to the `frontend-developer` subagent to:
   - Build React/Vue/Angular components
   - Implement responsive layouts
   - Setup TypeScript with strict mode
   - Create comprehensive tests (>85% coverage)
   - Ensure WCAG 2.1 AA accessibility compliance
   - Optimize bundle size
   - Implement state management
   - Setup build configuration

3. The frontend-developer will focus on:
   - Component architecture
   - TypeScript implementation
   - Accessibility compliance
   - Performance optimization
   - Test coverage
   - Responsive design
   - Build tooling
   - Documentation

4. Provide frontend implementation including:
   - Component code with TypeScript
   - Test files with >85% coverage
   - Storybook documentation
   - Performance metrics
   - Accessibility audit results
   - Bundle analysis
   - Build configuration
   - Migration guide if needed

## Output Format

Output frontend development work in Japanese, following the frontend-developer subagent's response format.

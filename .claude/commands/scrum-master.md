# Scrum Master Command

Get agile/scrum guidance using the scrum-master subagent.

## Agile/Scrum Topic

$ARGUMENTS

## Instructions

1. Understand the agile/scrum context:
   - Team situation and challenge
   - Current processes and metrics
   - Impediments and pain points
   - Stakeholder context
   - Maturity level and goals

2. Delegate to the `scrum-master` subagent to:
   - Facilitate team excellence
   - Analyze agile challenges
   - Design process improvements
   - Coach team members
   - Optimize ceremonies
   - Remove impediments
   - Improve metrics
   - Scale agile practices

3. The scrum-master will provide:
   - Team analysis and assessment
   - Process improvement recommendations
   - Impediment resolution strategies
   - Ceremony optimization
   - Coaching guidance
   - Metrics interpretation
   - Culture improvement ideas
   - Scaling frameworks advice

4. Provide agile guidance including:
   - Current state analysis
   - Root cause analysis
   - Recommended improvements
   - Implementation roadmap
   - Success metrics
   - Coaching approach
   - Team enablement plan
   - Celebration strategies

## Output Format

Output agile guidance in Japanese, following the scrum-master subagent's response format.

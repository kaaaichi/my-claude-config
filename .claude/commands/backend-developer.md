# Backend Developer Command

Execute backend development task using the backend-developer subagent.

## Backend Development Task

$ARGUMENTS

## Instructions

1. Analyze backend development requirements:
   - Task description and goals
   - Existing backend architecture
   - Database schema and models
   - API specifications
   - Performance requirements
   - Security constraints

2. Delegate to the `backend-developer` subagent to:
   - Develop RESTful APIs with proper HTTP semantics
   - Optimize database schemas and indexing
   - Implement authentication and authorization
   - Design caching strategies
   - Setup error handling and logging
   - Create comprehensive tests
   - Follow OWASP security guidelines
   - Achieve >80% test coverage

3. The backend-developer will work on:
   - Service implementation and structure
   - Database optimization
   - API endpoint design
   - Authentication/authorization
   - Caching and performance
   - Error handling
   - Testing and validation
   - Security best practices

4. Provide backend implementation including:
   - Implementation code
   - Database migrations
   - API documentation
   - Test suite
   - Performance metrics
   - Security analysis
   - Configuration guide
   - Deployment considerations

## Output Format

Output backend development work in Japanese, following the backend-developer subagent's response format.
